FlowInterpolator — IFNet

3-Level Coarse-to-Fine Pyramid
Bi-directional Optical Flow (t₀→t, t₁→t)
Differentiable Backward Warping
Learnable Blend Mask (Sigmoid)
PReLU Activations throughout
flow, mask, warped₀, warped₁ = IFNet(img₀, img₁, t=0.5) coarse = mask × warped₀ + (1-mask) × warped₁
02

RefinementNet — U-Net + CBAM

8-channel input (coarse + warps + flow + mask + originals)
CBAM: Channel + Spatial Attention
GroupNorm (stable with small batches)
Residual Learning with Learnable Scale
Tanh output → bounded corrections
residual = UNet(concat_inputs) × scale refined = clamp(coarse + residual, 0, 1)

THE PROJECT

Who built this?

Built for ISRO Hackathon 2025.

A two-stage deep learning pipeline that synthesizes intermediate satellite frames between two consecutive inputs — effectively doubling the temporal resolution of geostationary weather imagery without new hardware.

Trained on GOES-16. Tested on INSAT-3D.
